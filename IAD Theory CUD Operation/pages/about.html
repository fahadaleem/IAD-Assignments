About Contact App

This project is developed for the Internet Application Development (IAD) theory assignment, that contains the practice of JavaScript which includes Document Object Model (DOM), Window Object Model (WOM), arrays, loops, and variables. It contains all the basic Create, Update and Delete (CUD) Functionality

Features:

  • User can add contact
  • User can update contact
  • User can delete contact
  • User contact is saved in local storage to retain contacts for the long term.
